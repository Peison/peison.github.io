*Markdown*是一种可以使用普通文本编辑器编写的标记语言，通过类似HTML的标记语法，
它可以使普通文本内容具丰富多彩的格式。在平时的工作中和博客维护中需要使用，记下
平时常用的一些语法，以供查阅。
<!--more-->

##### 斜体
*It's just a test*
```
	*It's just a test*
```

##### 粗体
**It's just a test**
```
	**It's just a test**
```

##### 引用
> looke at it,it's funny 

```
	> looke at it,it's funny
```



##### 有序列表

2. b
3. c

```
1. a
2. b
3. c
```
